# v0.3.0 Phase 3 — Rust Toolchain Library

**Reader, Surface AST, Classifier, Diagnostics**

## Context

Phases 1-2 implemented all 19 JS surface macros. Phase 3 begins the Rust toolchain library — a modular library consumed by the compiler, linter, and formatter. This phase implements the first 4 of 6 modules defined in DD-20: reader, surface AST, classifier, and diagnostics. The macro expander, analysis, and kernel emitter are later phases.

New crate: `crates/lykn-lang` (library only, no binary). The existing `crates/lykn-cli` reader is too limited (no source locations, no keywords, no #-dispatch) — we build fresh.

## Module 3: Reader (`reader/`)

### Lexer (`reader/lexer.rs`)

Character-by-character tokenizer. Produces tokens with spans:

```rust
#[derive(Debug, Clone, PartialEq)]
pub enum Token {
    LParen,
    RParen,
    Dot,           // for dotted pairs
    Atom(String),
    Keyword(String),  // after stripping :
    String(String),
    Number(f64),
    Bool(bool),
    Null,          // null or undefined
    Hash,          // # dispatch trigger
    Quote,         // '
    Quasiquote,    // `
    Unquote,       // ,
    UnquoteSplice, // ,@
    Eof,
}

pub struct SpannedToken {
    pub token: Token,
    pub span: Span,
}
```

Lexer state: source chars, position, line/column tracking.

Key behaviors:

- `:foo` → `Token::Keyword("foo")`
- `true`/`false` → `Token::Bool`
- `null`/`undefined` → `Token::Null`
- `#` → `Token::Hash` (parser handles dispatch)
- `;` → skip to EOL
- Standard string escapes: `\n`, `\t`, `\\`, `\"`

### Parser (`reader/parser.rs`)

Consumes `SpannedToken` stream, produces `Vec<SExpr>`:

- `(...)` → `SExpr::List`
- `#a(...)` → `SExpr::List` with `Atom("array")` prepended
- `#o(...)` → `SExpr::List` with `Atom("object")` prepended
- `#;` → skip next form
- `#NNr...` → `SExpr::Number` with radix parsing
- `#|...|#` → skip (nestable block comment)
- `'expr` → `(quote expr)`
- `` `expr`` → `(quasiquote expr)`
- `,expr` → `(unquote expr)`
- `,@expr` → `(unquote-splicing expr)`
- `(a . b)` → `SExpr::Cons`

### Public API (`reader/mod.rs`)

```rust
pub fn read(source: &str) -> Result<Vec<SExpr>, ReadError>;
pub fn read_file(path: &Path) -> Result<Vec<SExpr>, ReadError>;
```

## Module 4: Typed Surface AST (`ast/surface.rs`)

Enum with variants for all surface forms. Each variant has structured fields with spans.

```rust
#[derive(Debug, Clone, PartialEq)]
pub struct TypeAnnotation {
    pub keyword: Spanned<String>,  // e.g., "number", "string", "any"
}

#[derive(Debug, Clone, PartialEq)]
pub struct TypedParam {
    pub type_ann: TypeAnnotation,
    pub name: Spanned<String>,
}

#[derive(Debug, Clone, PartialEq)]
pub struct FuncClause {
    pub args: Vec<TypedParam>,
    pub returns: Option<TypeAnnotation>,
    pub pre: Option<SExpr>,
    pub post: Option<SExpr>,
    pub body: Vec<SExpr>,
    pub span: Span,
}

#[derive(Debug, Clone, PartialEq)]
pub struct Constructor {
    pub name: Spanned<String>,
    pub fields: Vec<TypedParam>,
    pub span: Span,
}

#[derive(Debug, Clone, PartialEq)]
pub enum Pattern {
    Wildcard(Span),
    Literal(SExpr),
    Binding(Spanned<String>),
    Constructor { name: Spanned<String>, bindings: Vec<Pattern>, span: Span },
    Obj { pairs: Vec<(Spanned<String>, Pattern)>, span: Span },
}

#[derive(Debug, Clone, PartialEq)]
pub struct MatchClause {
    pub pattern: Pattern,
    pub guard: Option<SExpr>,
    pub body: Vec<SExpr>,
    pub span: Span,
}

#[derive(Debug, Clone, PartialEq)]
pub enum ThreadingStep {
    Bare(SExpr),        // bare symbol
    Call(Vec<SExpr>),   // list form
}

#[derive(Debug, Clone, PartialEq)]
pub enum SurfaceForm {
    Bind { name: SExpr, type_ann: Option<TypeAnnotation>, value: SExpr, span: Span },
    Func { name: Spanned<String>, clauses: Vec<FuncClause>, span: Span },
    Match { target: SExpr, clauses: Vec<MatchClause>, span: Span },
    Type { name: Spanned<String>, constructors: Vec<Constructor>, span: Span },
    Obj { pairs: Vec<(Spanned<String>, SExpr)>, span: Span },
    Cell { value: SExpr, span: Span },
    Express { target: SExpr, span: Span },
    Swap { target: SExpr, func: SExpr, extra_args: Vec<SExpr>, span: Span },
    Reset { target: SExpr, value: SExpr, span: Span },
    ThreadFirst { initial: SExpr, steps: Vec<ThreadingStep>, span: Span },
    ThreadLast { initial: SExpr, steps: Vec<ThreadingStep>, span: Span },
    SomeThreadFirst { initial: SExpr, steps: Vec<ThreadingStep>, span: Span },
    SomeThreadLast { initial: SExpr, steps: Vec<ThreadingStep>, span: Span },
    IfLet { pattern: Pattern, expr: SExpr, then_body: SExpr, else_body: Option<SExpr>, span: Span },
    WhenLet { pattern: Pattern, expr: SExpr, body: Vec<SExpr>, span: Span },
    Fn { params: Vec<TypedParam>, body: Vec<SExpr>, span: Span },
    Lambda { params: Vec<TypedParam>, body: Vec<SExpr>, span: Span },
    MacroDef { name: Spanned<String>, raw: SExpr, span: Span },
    ImportMacros { raw: SExpr, span: Span },
    KernelPassthrough { raw: SExpr, span: Span },
    FunctionCall { head: SExpr, args: Vec<SExpr>, span: Span },
}
```

### Public API (`classifier/mod.rs`)

```rust
pub fn classify(forms: &[SExpr]) -> Result<Vec<SurfaceForm>, Vec<Diagnostic>>;
pub fn classify_expr(expr: &SExpr) -> Result<SurfaceForm, Diagnostic>;
```

### Classification logic (`classifier/forms.rs`)

For each surface form, a `classify_*` function:

- `classify_bind(args: &[SExpr], span: Span) -> Result<SurfaceForm, Diagnostic>`
- `classify_func(args: &[SExpr], span: Span) -> Result<SurfaceForm, Diagnostic>`
- `classify_match(args: &[SExpr], span: Span) -> Result<SurfaceForm, Diagnostic>`
- etc.

Validation: reject malformed forms with source-located diagnostics. E.g.:

- `func` missing `:body` → error at func span
- `type` field without type keyword → error at field span
- `match` clause with 0 body forms → error at clause span
- `obj` with odd number of args → error
- `bind` with wrong arg count → error

## Module 6: Diagnostics (`diagnostics/`)

```rust
#[derive(Debug, Clone, PartialEq)]
pub enum Severity {
    Error,
    Warning,
    Info,
}

#[derive(Debug, Clone, PartialEq)]
pub struct Diagnostic {
    pub severity: Severity,
    pub message: String,
    pub span: Span,
    pub suggestion: Option<String>,
}

impl std::fmt::Display for Diagnostic { /* file:line:col: severity: message */ }
```

### S-Expression Serializer (`diagnostics/serializer.rs`)

```rust
pub fn serialize_sexpr(expr: &SExpr) -> String;
```

Converts SExpr tree back to readable s-expression text. Used for:

- Contract error messages: `"add: pre-condition failed: (> x 0)"`
- Diagnostic context in error messages
- Debug output
