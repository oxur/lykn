# v0.3.0 Phase 7 — CLI Integration & End-to-End Tests

## Context

Phases 3-6 built the complete `lykn-lang` Rust library (reader, AST, classifier, diagnostics, analysis, emitter, expander). Phase 7 wires it all together: a `lykn compile` CLI command and end-to-end integration tests that verify the Rust pipeline produces output identical to the JS pipeline.

## What to Implement

### 1. `lykn compile` command in `crates/lykn-cli`

Add a `compile` subcommand to the existing CLI binary. The existing CLI uses manual arg parsing with `fmt` and `check` commands.

**Options:**

- `lykn compile <file.lykn>` — compile to JS, output to stdout
- `lykn compile <file.lykn> -o output.js` — write to file
- `lykn compile <file.lykn> --strip-assertions` — omit type checks/contracts
- `lykn compile <file.lykn> --kernel-json` — output kernel JSON (skip JS compilation)

**Pipeline:**

```
read(source) → expand(forms) → classify(expanded) → analyze(classified)
  → if errors: print diagnostics to stderr, exit 1
  → emit(classified, registry, strip) → kernel SExpr
  → if --kernel-json: output JSON, done
  → bridge to JS kernel compiler (deno) → JS output
```

**JS kernel compiler bridge:**

- Write kernel JSON to temp file
- Run `deno eval` with script that imports `compile` from `src/compiler.js`, reads JSON, parses to AST, compiles, outputs JS
- Capture stdout → JS output

### 3. CLI Architecture

Migrate from manual arg parsing to `clap` derive:

```rust
use clap::{Parser, Subcommand};

#[derive(Parser)]
#[command(name = "lykn", version, about = "lykn language toolchain")]
struct Cli {
    #[command(subcommand)]
    command: Commands,
}

#[derive(Subcommand)]
enum Commands {
    /// Format .lykn files
    Fmt {
        files: Vec<PathBuf>,
        #[arg(short, long)]
        write: bool,
    },
    /// Check .lykn syntax
    Check {
        files: Vec<PathBuf>,
    },
    /// Compile .lykn to JavaScript
    Compile {
        file: PathBuf,
        #[arg(short, long)]
        output: Option<PathBuf>,
        #[arg(long)]
        strip_assertions: bool,
        #[arg(long)]
        kernel_json: bool,
    },
}
```

### 4. Compile Pipeline (`compile.rs`)

```rust
pub fn compile_file(
    path: &Path,
    strip_assertions: bool,
    kernel_json_only: bool,
) -> Result<String, Vec<Diagnostic>>;
```

Steps:

1. Read source file
2. Parse with `lykn_lang::reader::read()`
3. Expand macros with `lykn_lang::expander::expand()`
4. Classify with `lykn_lang::classifier::classify()`
5. Analyze with `lykn_lang::analysis::analyze()`
6. If `has_errors`: return diagnostics
7. Emit with `lykn_lang::emitter::emit()`
8. If `kernel_json_only`: return JSON via `emit_module_json()`
9. Bridge to JS kernel compiler → return JS output

### 5. JS Kernel Compiler Bridge (`bridge.rs`)

```rust
pub fn kernel_json_to_js(kernel_json: &str) -> Result<String, LyknError>;
```

Implementation:

```javascript
// Script passed to `deno eval`
import { compile } from "./src/compiler.js";
const kernel = JSON.parse(Deno.readTextFileSync(Deno.args[0]));
// Convert JSON arrays back to AST nodes
function fromJson(val) {
    if (Array.isArray(val)) return { type: "list", values: val.map(fromJson) };
    if (typeof val === "string") return { type: "atom", value: val };
    if (typeof val === "number") return { type: "number", value: val };
    if (typeof val === "boolean") return { type: "atom", value: String(val) };
    if (val === null) return { type: "atom", value: "null" };
    return val;
}
const ast = kernel.map(fromJson);
console.log(compile(ast));
```

Write kernel JSON to temp file, pass path as arg to `deno eval`.

### 6. End-to-End Test Programs

Four `.lykn` programs in `test/fixtures/e2e/`:

**`shapes.lykn`** — ADTs + pattern matching:

```lisp
(type Shape (Circle :number radius) (Rect :number width :number height) Point)
(func area
  :args (:any shape)
  :returns :number
  :body (match shape
    ((Circle r) (* 3.14159 (* r r)))
    ((Rect w h) (* w h))
    (Point 0)))
```

**`option-threading.lykn`** — Option + some->:

```lisp
(type Option (Some :any value) None)
(func find-user :args (:number id) :returns :any
  :body (if (=== id 1) (Some "Duncan") None))
(bind greeting (some-> (find-user 1) (+ "Hello, ")))
```

**`contracts.lykn`** — func with contracts:

```lisp
(func safe-divide
  :args (:number a :number b)
  :returns :number
  :pre (!== b 0)
  :body (/ a b))
```

**`cells.lykn`** — cells + mutation:

```lisp
(bind counter (cell 0))
(swap! counter (=> (n) (+ n 1)))
(console:log (express counter))
```

### 8. E2E Test Runner

```rust
// For each E2E test:
// 1. Compile with Rust: read → expand → classify → analyze → emit → kernel JSON
// 2. Compile with JS: deno eval using lykn(source) → expand → kernel JSON

// 4. If kernel JSON matches, run JS kernel compiler on both → should produce same JS
// 5. Run the produced JS in Deno, verify it executes without error
```
